Qué vamos a ver

El primer lenguaje que vamos a ver es HTML, que sirve para dar estructura a la página web. Es un lenguaje marcado, es decir, funciona mediante unas estructuras llamadas etiquetas: párrafo, título, sección, etc.

El segundo lenguaje que conoceremos se llama CSS, y sirve para dar estética a la página web. Funciona mediante criterios de importancia, especificidad y orden. No usa etiquetas, sino declaraciones.

El tercer lenguaje es JavaScript que sirve para dar una interfaz interactiva. Así, para poder hacer un login funcional, por ejemplo, se necesitaria este lenguaje.
